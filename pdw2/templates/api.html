API - Calculateur du domaine public

L’interface de programmation (API) du Calculateur du Domaine Public vous permet de calculer le statut juridique d’une œuvre. Il s’appuie sur l’arbre de décisions représentant le régime du droit d’auteur en France développé par l’IViR, Kennisland et la Bibliothèque nationale du Luxembourg dans le cadre du projet EuropeanaConnect.

L'API est disponible à cette adresse :/api/pd

Pour faire une requête vous devez fournir les paramétres suivants :

  • jurisdiction : le pays pour lequel vous souhaitez calculer le statut juridique de l'oeuvre : 'austria', 'belgium', 'bulgaria', 'cyprus', 'czech', 'denmark', 'estonia', 'finland', 'france', 'germany', 'greece', 'hungary', 'iceland', 'ireland', 'italy', 'latvia', 'lithuania', 'luxembourg', 'malta', 'netherlands', 'norway', 'poland', 'portugal', 'romania', 'slovakia', 'spain', 'sweden', 'switzerland', 'uk'.
  • work: détails de l'oeuvre (au format json ou au format rdf).

La réponse contenant les résultats du Calculateur sera retournée au format JSON

Exemple de requête

/api/pd?jurisdiction=france/bnf&work=http://data.bnf.fr/15533097/2046___film/rdf.xml&detail=medium&lang=fr

Résultat du Calculateur

    {
      "output": "The PDCalc reasoner output string...",
      "work": A JSON string of the input used,
      "error: "Error messages from the reasoner" 
    }
